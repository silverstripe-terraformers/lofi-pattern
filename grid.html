Summary

The grid is based on a system of 3 breakpoints: Desktop, Tablet and Mobile.

Desktop

Desktop is a 4 card layout from left to right with one .row class div containing all cards. The .card class is the immediate child of the Materialize.css "col" class div: class="col l3 m6 s12". Actual code example here, re-size screen to see breakpoint demonstration.
